# ProfJohns website design kit (low/mid-fi)

19 marketing-site section mockups in the product's Swiss system — flat, zero
gradients, 1px borders, ink + teal. Skeleton bars stand in for copy; only key
actions carry real text. Every product depiction mirrors the actual UI
(canvas nodes, Compose, Watch, Agents, semantic/figure search) — nothing
invented.

- Regenerate: `node design/website-kit/generate.mjs`
- Preview all: open `gallery.html`
- Figma: drag any `.svg` into Figma — it imports as editable vectors.
  (A generated file also exists: "ProfJohns Website Kit — low-fi sections".)

Sections: nav, hero, social proof, process (Scout->Synthesize->Compose),
canvas feature, Compose feature (canvas->paper), traceable citations, agents,
Watch, semantic+figure search, voice training, results band, comparison,
testimonials, pricing, FAQ, CTA, footer, atoms sheet.

Deliberate choices: no stock photography at this fidelity (photos fight the
flat system and the "only our UI" rule) — imagery slots come in the hi-fi
pass, along with Hyperframes product animations.
